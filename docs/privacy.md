# 개인정보처리방침

**마지막 업데이트: 2026년**

## 개요

Link Collector (이하 "확장 프로그램")는 사용자의 프라이버시를 최우선으로 생각합니다. 본 문서는 확장 프로그램이 어떤 데이터를 수집하고, 사용하며, 보호하는지를 설명합니다.

---

## 1. 수집 데이터

### 1.1 수집하는 데이터

#### 로컬 저장 데이터
- **사용자 설정**:
  - 단축키 선택
  - 열기 방식 (새 탭/백그라운드 탭/새 창)
  - 최대 탭 수
  - 선택 박스 색상
  - 도메인 필터 설정
  - 확인 다이얼로그 토글

**저장 위치**: Chrome 동기화 저장소 (개인 기기에만 저장, 암호화됨)

### 1.2 수집하지 않는 데이터

❌ 개인 정보 (이름, 이메일, 전화번호 등)
❌ 방문한 웹사이트 목록
❌ 브라우징 기록
❌ 쿠키 또는 추적 데이터
❌ 사용자가 열려고 하는 링크 URL (임시로만 처리)
❌ 위치 정보
❌ 연락처 또는 캘린더 정보

---

## 2. 데이터 사용 목적

### 2.1 설정 저장
사용자가 선택한 옵션을 저장하여 다음 사용 시 동일한 설정을 적용합니다.

### 2.2 기능 제공
- 선택 박스 색상 표시
- 단축키 감지 및 동작
- 링크 필터링 (도메인별)
- 탭 열기 방식 적용

---

## 3. 데이터 공유

### 3.1 외부 공유
❌ **제3자와 공유하지 않습니다**
- 서버에 데이터 전송 없음
- 광고 네트워크와 공유 없음
- 분석 도구 없음
- 소셜 미디어 통합 없음

### 3.2 내부 처리
데이터는 순수하게 로컬 브라우저에서만 처리됩니다.

---

## 4. 데이터 보안

### 4.1 저장소 보안
- Chrome의 기본 암호화된 저장소 사용
- 개별 사용자 계정으로 보호
- 동기화 데이터는 Google 암호화 프로토콜로 보호

### 4.2 통신 보안
❌ 네트워크 통신 없음 (서버 요청 없음)
✅ 모든 처리는 브라우저 내에서만 수행

### 4.3 코드 보안
- Manifest V3 준수 (최신 보안 표준)
- Content Security Policy (CSP) 적용
- XSS 방지 대책 포함

---

## 5. 사용자 권리

### 5.1 데이터 접근
사용자는 언제든지 다음을 통해 자신의 설정을 확인할 수 있습니다:
- Chrome 주소창: `chrome://sync/` (동기화 상태 확인)

### 5.2 데이터 수정 및 삭제
사용자는 언제든지:
1. **설정 변경**: 팝업 창에서 모든 설정 수정 가능
2. **데이터 삭제**:
   - 확장 프로그램 제거 → 관련 모든 데이터 자동 삭제
   - Chrome 초기화 → 모든 확장 데이터 삭제

---

## 6. Chrome 권한 설명

### 6.1 `tabs` 권한
**용도**: 새 탭 생성 및 관리
**수집 정보**: 없음 (탭 생성만 수행)

### 6.2 `storage` 권한
**용도**: 사용자 설정 저장 및 로드
**수집 정보**: 위의 1.1절 참조

### 6.3 `content_scripts` (모든 웹사이트)
**용도**: Z 키 감지 및 링크 수집 (로컬에서만)
**수집 정보**: 없음 (링크 URL은 메모리에서만 임시 처리)

---

## 7. 제3자 서비스

❌ **외부 서비스 통합 없음**
- 분석 도구 (Google Analytics, Mixpanel 등): 없음
- 광고 네트워크: 없음
- 클라우드 서비스: 없음
- 소셜 미디어 플러그인: 없음

---

## 8. 쿠키 및 추적

❌ 쿠키 사용 없음
❌ 추적 픽셀 없음
❌ 핑거프린팅 없음
❌ 광고 추적 없음

---

## 9. 어린이 개인정보 보호

- 본 확장 프로그램은 13세 미만의 어린이를 대상으로 하지 않습니다.
- 의도적으로 어린이 정보를 수집하지 않습니다.

## 12. 법적 준수

### 12.1 GDPR (유럽 연합)
✅ 준수
- 개인 데이터 최소화
- 명확한 정책 제공
- 언제든 데이터 삭제 가능
- 제3자 공유 없음

### 12.2 CCPA (캘리포니아)
✅ 준수
- 개인 정보 수집 없음
- 기술적 데이터만 저장
- 사용자 통제권 제공

### 12.3 기타 법규
- 적용되는 모든 국가 법규 준수
- 해당 지역에서 강제되는 변경 시 즉시 적용

---

## 13. 데이터 보존

- **저장 기간**: 확장 프로그램 사용 중에만
- **삭제 시기**: 확장 프로그램 제거 시 자동 삭제
- **백업**: 없음 (Chrome 동기화 제외)

---

## 14. 투명성 및 통제

사용자는 다음을 통해 완전한 통제권을 가집니다:

1. **설정 확인**: 팝업 창에서 저장된 모든 설정 확인
2. **동기화 비활성화**: Chrome 설정에서 확장 프로그램 동기화 끄기
3. **완전 제거**: 확장 프로그램 제거 시 모든 데이터 삭제
